# Conventions

## Scope

Этот проект остается простым статическим сайтом без сборки и без фреймворков.

## Structure

- `index.html` содержит только структуру страницы.
- `styles.css` содержит все стили проекта.
- JavaScript не добавляем, пока он не нужен для реального поведения.
- Локальные шрифты храним в `assets/fonts`.

## Typography

- Используем только шрифты из Google Fonts с открытой лицензией.
- В проекте подключаем эти шрифты локально через `@font-face`, а не через внешний `<link>` на Google Fonts.
- `Geo` используем для заголовков и крупных акцентов.
- `Sometype Mono` используем только для моноширинных элементов: подписи, коды, технические метки, размеры.
- Если нужен новый шрифт для основного текста, он тоже должен быть из Google Fonts с открытой лицензией и согласован отдельно.

## Layout

- Основной лист фиксированного размера: `400px x 800px`.
- Лист всегда центрируется по горизонтали.
- Внутренняя рамка строится как отдельный блок внутри листа.
- Не делаем резиновую сетку, если это не согласовано отдельно.

## HTML

- Используем семантические теги: `main`, `section`, `header`, `nav`, `footer`.
- Имена классов только в `kebab-case`.
- Не используем инлайновые стили.
- Не добавляем лишние обертки без причины.

## CSS

- Стили пишем от общего к частному.
- Для повторяющихся значений выносим переменные в `:root`, когда таких значений становится несколько.
- Цвета, отступы и рамки держим минимальными и спокойными.
- Анимации и эффекты добавляем только если они поддерживают композицию.

## Visual Direction

- База: чистый лист, много воздуха, строгая геометрия.
- Основной акцент на рамке, пропорциях и отступах.
- Декор добавляем только после утверждения базовой композиции.
- Палитра ограничена: `#ffffff`, `#000000`, `#9999ff`, `#ffcc99` и оттенки серого.
- Яркие акценты используем дозированно, а не как фон по умолчанию.

## Editing Rules

- Каждое изменение должно сохранять минимализм макета.
- Новые блоки добавляем только если понятна их функция.
- Сначала согласуем композицию, потом типографику, потом контент.
